My works

Musiverse - A music streaming website made using HTML, CSS & Javascript.

The MusiVerse web development project aims to create a user-friendly music streaming platform, offering seamless song playback, and a dynamic user interface. The user can multitask by, for example, working on one thing while listening to music on another. The project's primary objectives include designing an intuitive front-end interface,enabling smooth audio streaming while being ad-free.
